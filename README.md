# Reja Application

Reja is simple entry level a Node.js-based backend service connected to an external database hosted on a VPS.
The application runs locally on port **4009** and communicates  with a remote production database.

“This project was developed to practice and demonstrate CRUD (Create, Read, Update, Delete) operations.”
---

## Overview

This project is structured as a backend service designed to:

* Connect to an external VPS-hosted database
* Provide REST API endpoints
* Run locally for development

---

## Technology Stack

* Node.js
* Express
* Mongodb
* External Database (VPS Hosted)

---

## Installation

Clone the repository:

```bash
git clone <repository-url>
cd reja
```

Install dependencies:

```bash
npm install
```

---

## Running the Application

Start the development server:

```bash
npm run dev
```

Or start normally:

```bash
npm start
```

The server will run on:

```
http://localhost:4009
```
